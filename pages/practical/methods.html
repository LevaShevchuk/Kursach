Разбор типового задания

Условие задачи:

Разработать метод, возвращающий произведение трёх чисел. Входные параметры метода – три целых числа.

Программный код:

// Блок основной программы
Console.WriteLine("Введите первое число: ");
int a = Convert.ToInt32(Console.ReadLine());

Console.WriteLine("Введите второе число: ");
int b = Convert.ToInt32(Console.ReadLine());

Console.WriteLine("Введите третье число: ");
int c = Convert.ToInt32(Console.ReadLine());

int p = Proiz(a, b, c);
Console.WriteLine("Произведение чисел равно " + p);

Console.ReadKey();

// блок определения метода, возвращающего произведение чисел
static int Proiz(int x, int y, int z)
{
    return x * y * z;
}

Пример работы программы:

Введите первое число:
5
Введите второе число:
6
Введите третье число:
3
Произведение чисел равно 90
