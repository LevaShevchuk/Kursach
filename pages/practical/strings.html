Работа со строками

Разбор типового задания

Условие задачи:

Дана строка. Определить, сколько в ней символов r, k, t.

Программный код:

Console.WriteLine("Введите строку из символов: ");
string s = Console.ReadLine();

int r = 0;
int k = 0;
int t = 0;
for (int i = 0; i < s.Length; i++)
{    
    if (s[i] == 'r')
    {
        r++;
    }

    if (s[i] == 'k')
    {
        k++;
    }
    if (s[i] == 't')
    {
        t++;
    }
}

if (r == 0)
{
    Console.WriteLine("Cимволы r в введённой строке отсутствуют!");
}

else
{
    Console.WriteLine("Количество символов r в введённой строке равно " + r);
}


if (k == 0)
{
    Console.WriteLine("Cимволы k в введённой строке отсутствуют!");
}

else
{
    Console.WriteLine("Количество символов k в введённой строке равно " + k);
}


if (t == 0)
{
    Console.WriteLine("Cимволы t в введённой строке отсутствуют!");
}

else
{
    Console.WriteLine("Количество символов t в введённой строке равно " + t);
}
Console.ReadKey();

Пример работы программы:

Введите строку из символов:
rgrgkgkgtgtg
Количество символов r в введённой строке равно 2
Количество символов k в введённой строке равно 2
Количество символов t в введённой строке равно 2

Задания для самостоятельного решения

Задание 1:

Дана строка. Преобразовать её, поместив вначале все символы, расположенные на чётных позициях строки, а затем все символы, расположенные на нечётных позициях.

Задание 2:

Дана исходная строка цифр без пробелов. Чётные цифры записать в одну строку, а нечётные – в другую. Вывести исходную строку и полученные строки.

Задание 3:

Дана строка, содержашая английский текст. Определить количество слов, начинающихся с буквы b.
