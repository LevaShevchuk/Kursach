Рекурсивные функции

Разбор типового задания

Условие задачи:

Дано натуральное число N. Вычислите сумму его цифр. Использовать рекурсивную функцию.

Программный код:

// блок определения рекурсивной функции
static int summatsifr(int n)
{
    if (n < 10)
    {
        return n;
    }
    else
    {
        return n % 10 + summatsifr(n / 10);
    }
}
      
// блок основной программы
Console.WriteLine("Введите натуральное число: ");
int x = Convert.ToInt32(Console.ReadLine());
int sum = summatsifr(x);
Console.WriteLine("Сумма цифр числа " + x + " равна " + sum);
