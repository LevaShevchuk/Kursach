Разбор типового задания

Условие задачи:

Разработать программу, реализующую эпизод применения компьютера в книжном магазине. Компьютер запрашивает стоимость книги, сумму денег, внесённую покупателем. Если сдача не требуется, на экране печатается сообщение «Спасибо!». Если денег внесено больше, то печатает «Возьмите сдачу» и указывает сумму сдачи. Если денег недостаточно, то печатает об этом сообщение и указывает размер недостающей суммы.

Программный код:

Console.WriteLine("Введите стоимость книги: ");
decimal x = Convert.ToDecimal(Console.ReadLine());

Console.WriteLine("Введите внесённую сумму денег: ");
decimal y = Convert.ToDecimal(Console.ReadLine());

if (x == y)
{
   Console.WriteLine("Спасибо!");
}

else if (x < y)
{
   decimal z = y - x;
   Console.WriteLine("Возьмите сдачу в размере " + Math.Round(z, 2) + " рублей");
}

else
{
   decimal z = x - y;
   Console.WriteLine("Недостаточно денег! Необходимо внести ещё " + Math.Round(z, 2) + " рублей");
}
Console.ReadKey();

Пример работы программы:

Введите стоимость книги:
590
Введите внесённую сумму денег:
600
Возьмите сдачу в размере 10 рублей
