Консольный ввод-вывод

Разбор типового задания

Условие задачи:

Разработать программу вывода на экран в одну строку четырёх чисел, введённых пользователем с клавиатуры, с одним пробелом между ними.

Программный код:

Console.WriteLine("Введите первое число: ");
double a = Convert.ToDouble(Console.ReadLine());

Console.WriteLine("Введите второе число: ");
double b = Convert.ToDouble(Console.ReadLine());

Console.WriteLine("Введите третье число: ");
double c = Convert.ToDouble(Console.ReadLine());

Console.WriteLine("Введите четвёртое число: ");
double d = Convert.ToDouble(Console.ReadLine());

Console.WriteLine("Введённые Вами числа: ");
Console.WriteLine($"{a} {b} {c} {d}");
Console.ReadKey();

Пример работы программы:

Введите первое число:
9
Введите второе число:
14
Введите третье число:
14,9
Введите четвёртое число:
9,14
Введённые Вами числа:
9 14 14,9 9,14

Задания для самостоятельного решения

Задание 1:

Разработать программу вывода на экран числа, вводимого пользователем с клавиатуры. Выводимому числу должно предшествовать сообщение «Вы ввели число».

Задание 2:

Разработать программу вывода на одной строке трёх чисел, введённых пользователем с клавиатуры.

Задание 3:

Разработать программу вывода на экран «столбиком» четырёх слов, введённых пользователем с клавиатуры.
