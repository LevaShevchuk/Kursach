Разбор типового задания

Условие задачи:

Процентная ставка по вкладу — 15% годовых. Первоначальный вклад составил А р. Выведите на экран, через сколько лет он станет больше В р.

Программный код:

Console.WriteLine("Введите сумму первоначального вклада в рублях: ");
double a = Convert.ToDouble(Console.ReadLine());

Console.WriteLine("Введите конечную сумму вклада: ");
double b = Convert.ToDouble(Console.ReadLine());

int k = 0;
double x = a;

do
{
   x = x + x * 0.15;
   k++;
   Console.WriteLine("Через " + k + " год(а)/лет сумма вклада составит " + Math.Round(x, 2) + " рублей");
}
while (x <= b);

Console.WriteLine("Количество лет, через которое сумма вклада превысит " + b + " рублей: " + k);
Console.ReadKey();
