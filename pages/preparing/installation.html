Установка Visual Studio 2022 Community

Для создания приложения на языке C# необходимы:

  • Текстовый редактор, в котором была бы возможность напечатать код программы;
  • Компилятор, который бы скомпилировал набранный в текстовом редакторе программный код в приложение exe;
  • Фреймворк .NET, который требуется для компиляции и выполнения программы.

Для того, чтобы облегчить написание, а также тестирование и отладку программного кода обычно используют специальные среды разработки, в частности, Visual Studio.

Для создания приложений на C# в рамках данного электронного сборника используется бесплатная и полнофункциональная среда разработки Visual Studio Community 2022.

После загрузки необходимо запустить программу установщика. В открывшемся окне будет предложено выбрать те компоненты, которые можно установить вместе с Visual Studio. Следует отметить, что Visual Studio - очень функциональная среда разработки и позволяет разрабатывать приложения с помощью множества языков и платформ. В наборе рабочих нагрузок необходимо выбрать пункт «Разработка классических приложений .NET». Можно выбрать больше опций или вообще все опции, однако стоит учитывать свободный размер на жёстком диске - чем больше опций будет выбрано, соответственно, тем больше места на диске будет занято.

При инсталляции Visual Studio на компьютер будут установлены все необходимые инструменты для разработки программ, в том числе фреймворк .NET 5.

После завершения установки можно перейти к созданию первой программы. Вначале необходимо открыть Visual Studio. На стартовом экране нужно выбрать пункт «Создание проекта».

В следующем окне в качестве типа проекта следует выбрать пункт «Консольное приложение».

Далее в поле «Имя пректа» необходимо дать проекту какое-либо название. Например, HelloApp. А также указать папку, где будет располагаться проект. После чего необходимо нажать «Далее».

В появившемся окне необходимо выбрать в меню раскрывающегося списка актуальную версию .NET «.Net 7.0 (Поддержка со стандартным сроком)» и нажать «Создать».

После этого Visual Studio создаст и откроет проект:

В большом поле, расположенном в центре и представляющим собой текстовый редактор, находится сгенерированный по умолчанию программный код C#.

Справа находится окно Обозревателя решений, в котором можно увидеть структуру проекта. В данном случае в окне отображается сгенерированная по умолчанию структура: узел Зависимости - это узел содержит сборки dll, которые добавлены в проект по умолчанию. Эти сборки содержат классы библиотеки .NET, которые будет использовать C#. Однако не всегда все сборки нужны. Ненужные потом можно удалить, в то же время если понадобится добавить какую-нибудь нужную библиотеку, то именно в этот узел она будет добавляться.

Далее представлен непосредственно сам файл программного кода программы Program.cs. Как раз этот файл и открыт в центральном окне. В данный момент в файле содержится 2 строчки программного кода:

// See https://aka.ms/new-console-template for more information
Console.WriteLine("Hello, World!")

Теперь можно запустить на выполнение с помощью клавиши F5 или с Панели инструментов, нажав зелёную стрелочку. В результате выполнения данной программы в консоли должна отобразиться строчка Hello, World!

Hello, World!

Теперь необходимо изменить весь программный код на следующий:

Console.WriteLine("Введите своё имя: ");
string name = Console.ReadLine();         // вводим своё имя
Console.WriteLine($"Привет, {name}!");    // выводим имя на консоль

Метод Console.Write() используется для вывода на консоль некоторой строки. В данном случае выводится строчка "Введите своё имя: "

Console.WriteLine("Введите своё имя: ");

На второй строке определяется строковая переменная name, в которую пользователь вводит информацию с консоли:

var name = Console.ReadLine();

Ключевое слово var применяется для определения переменных. В данном случае переменная называется name. И ей присваивается результат метода Console.ReadLine(), при помощи которого можно считать с консоли строку и записать её в значение переменной name.

Затем введённое имя выводится на консоль:

Console.WriteLine($"Привет, {name}!");

Для того, чтобы ввести значение переменной name внутрь выводимой на консоль строки, применяются фигурные скобки {}. При выводе строки на консоль выражение {name} будет заменяться на значение переменной name - введённое имя.

Однако для того, чтобы можно было вводить таким образом значения переменных внутрь строки, перед строкой указывается знак доллара $.

Теперь необходимо протестировать проект, запустив его на выполнение, нажав клавишу F5 или зелёную стрелочку на панели Visual Studio.
