Циклы

Циклы являются управляющими конструкциями, позволяя в зависимости от определённых условий выполнять некоторое действие множество раз. В C# имеются следующие виды циклов:

  • for;
  • do...while;
  • while;
  • foreach.

Цикл for

Цикл for имеет следующее формальное определение:

for ([инициализация счетчика]; [условие]; [изменение счетчика])
{
    // действия
}

Далее рассматривается стандартный цикл for:

for (int i = 0; i < 9; i++)
{
     Console.WriteLine($"Квадрат числа {i} равен {i*i}");
}

Первая часть объявления цикла - int i = 0 - создаёт и инициализирует счётчик i. Счётчик необязательно должен представлять тип int. Это может быть и другой числовой тип, например, float. Перед выполнением цикла его значение будет равно 0. В данном случае это то же самое, что и объявление переменной.

Вторая часть - условие, при котором будет выполняться цикл. Пока условное выражение возвращает true, цикл будет выполняться. В данном случае цикл будет выполняться, пока счётчик i не достигнет 9.

И третья часть - приращение счётчика на единицу. Необязательно увеличивать переменную i на единицу. Её также можно и уменьшать, например, i--

В итоге блок цикла сработает 9 раз, пока значение i не станет равным 9. И каждый раз это значение будет увеличиваться на 1.

Необязательно указывать все условия при объявлении цикла. Например, можно написать так:

int i = 0;
for (; ;)
{
     Console.WriteLine($"Квадрат числа {i} равен {i*i}");
}

Формально определение цикла осталось тем же, только теперь блоки в определении пустые: for (; i <;). В данном случае нет инициализированной переменной-счётчика, нет условия, поэтому цикл будет работать вечно.

Также можно опустить ряд блоков:

int i = 0;
for (; i < 9;)
{
     Console.WriteLine($"Квадрат числа {i} равен {i*i}");
}

Данный пример эквивалентен первому примеру: здесь также есть счётчик, только создан он вне цикла. Присутствует условие выполнения цикла. И есть приращение счётчика уже в самом блоке for.

Цикл do...while

В цикле do...while сначала выполняется код цикла, а потом происходит проверка условия в инструкции while. И пока это условие истинно, цикл повторяется. Например:

int i = 6;
do
{
     Console.WriteLine(i);
     i--
}
while (i > 0);

В данном случае код цикла сработает 6 раз, пока i не станет равным нулю. Важно отметить, что цикл do гарантирует хотя бы единократное выполнение действий, даже если условие в инструкции while не будет истинно. Программный код можно изменить следующим образом:

int i = -1;
do
{
     Console.WriteLine(i);
     i--
}
while (i > 0);

Цикл while

В отличие от цикла do...while цикл while сразу проверяет истинность условия, и если оно истинно, то код цикла выполняется:

int i = 6;
while (i > 0)
{
   Console.WriteLine(i);
   i--;
}

Операторы continue и break

Иногда возникает ситуация, когда требуется выйти из цикла, не дожидаясь его завершения. В этом случае можно воспользоваться оператором break. Например:

for (int i = 0; i < 9; i++)
{
      if (i == 5)
            break;
       Console.WriteLine(i);
}

Хотя в условии цикла сказано, что цикл будет выполняться, пока счётчик i не достигнет значения 9, в реальности цикл сработает 5 раз. Так как при достижении счётчиком i значения 5, сработает оператор break, и цикл завершится.

0
1
2
3
4

Но что делать, если необходимо, чтобы при проверке цикл не завершался, а просто пропускал текущую итерацию. Для этого можно воспользоваться оператором continue:

for (int i = 0; i < 9; i++)
{
      if (i == 5)
            continue;
       Console.WriteLine(i);
}

В этом случае цикл, когда дойдет до числа 5, которое не удовлетворяет условию проверки, просто пропустит это число и перейдет к следующей итерации:

0
1
2
3
4
6
7
8
9

Цикл foreach

Цикл foreach предназначен для перебора элементов в контейнерах, в том числе в массивах. Формальное объявление цикла foreach:

foreach (тип_данных название_переменной in контейнер)
{
    // действия
}

Например:

int[] numbers = new int[] { 1, 2, 3, 4, 5 };
foreach (int i in numbers)
{
    Console.WriteLine(i);
}

Консольный вывод:

1
2
3
4
5

Здесь в качестве контейнера выступает массив данных типа int. Поэтому была объявлена переменная с типом int.

Подобные действия можно совершить и с помощью цикла for:

int[] numbers = new int[] { 1, 2, 3, 4, 5 };
for (int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine(i);
}

В то же время цикл for более гибкий по сравнению с foreach. Если foreach последовательно извлекает элементы контейнера и только для чтения, то в цикле for можно перескакивать на несколько элементов вперёд в зависимости от приращения счётчика, а также можно изменять элементы:

int[] numbers = new int[] { 1, 2, 3, 4, 5 };
for (int i = 0; i < numbers.Length; i++)
{
    numbers[i] = numbers[i] * 2;
    Console.WriteLine(i);
}

Консольный вывод:

0
1
2
3
4
