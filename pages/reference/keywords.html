Ключевые слова

Ключевые слова — предварительно определённые зарезервированные идентификаторы, которые имеют специальные значения для компилятора.

Глоссарий ключевых слов


abstract — указывает, что изменяемый элемент имеет отсутствующую или неполную реализацию. Модификатор abstract можно использовать с классами, методами, свойствами, индексаторами и событиями.

as — явным образом преобразует результат выражения в указанный ссылочный или поддерживающий значения NULL тип. Если такое преобразование невозможно, оператор as возвращает значение null.

base — используется для доступа к членам базового класса из производного.

bool — псевдоним для типа структуры System.Boolean .NET, представляющий логическое значение: true или false.

break — завершает выполнение ближайшего оператора внешнего цикла или switch, в котором он находится. Управление передаётся оператору, который расположен после завершённого оператора.

byte — тип данных, хранящий целое число от 0 до 255 и занимающий 1 байт. Представлен системным типом System.Byte.

case — оператор выбора, используемый в условной конструкции switch/case.

catch — задаёт обработчик для различных исключений.

char — хранит одиночный символ в кодировке Unicode и занимает 2 байта. Представлен системным типом System.Char.

checked — используется для явного включения проверки переполнения при выполнении арифметических операций и преобразований с данными целого типа.

class — используется для объявления классов.

const — используется для объявления константного поля или константной локальной.

continue — передаёт управление следующей итерации вложенного оператора while, do, for или foreach, в котором она встречается.

decimal — тип данных, хранящий десятичное дробное число. Если употребляется без десятичной запятой, имеет значение от ±1.0 * 10-28 до ±7.9228 * 1028, может хранить 28 знаков после запятой и занимает 16 байт. Представлен системным типом System.Decimal.

default — используется для указания метки по умолчанию в инструкции switch. Также ключевое слово default применяется в качестве оператора или литерала по умолчанию для создания значения по умолчанию для типа.

delegate — имеет ссылочный тип, который можно использовать для инкапсуляции именованного или анонимного метода.

do — выполняет оператор или блок операторов, пока определенное логическое выражение равно значению true.

double — тип данных, хранящий число с плавающей точкой от ±5.0 * 10-324 до ±1.7 * 10308 и занимающий 8 байта. Представлен системным типом System.Double.

else — иначе.

enum — тип значения, определённый набором именованных констант применяемого целочисленного типа.

event — используется для объявления события в классе Publisher.

explicit — используется для явного преобразования типов.

extern — используется для объявления метода с внешней реализацией.

false — ложь.

finally — позволяет выполнить очистку всех ресурсов, выделенных в блоке try, и запускать программный код даже при возникновении исключения в блоке try.

fixed — задаёт указатель на управляемую переменную и «закрепляет» эту переменную во время выполнения оператора.

float — тип данных, хранящий число с плавающей точкой от -3.4 * 1038 до 3.4 * 1038 и занимающий 4 байта. Представлен системным типом System.Single.

for — выполняет оператор или блок операторов, пока определённое логическое выражение равно значению true.

foreach — выполняет оператор или блок операторов для каждого элемента в экземпляре типа, который реализует интерфейс System.Collections.IEnumerable или System.Collections.Generic.IEnumerable<T>.

goto — передаёт управление программой непосредственно оператору с меткой.

if — определяет, какой оператор будет выполняться при выполнении условия, заданного логическим выражением.

implicit — используется для неявного преобразования типов.

in — вызывает передачу аргументов по ссылке, но гарантирует, что аргумент не будет изменен. Также ключевое слово in указывает, что параметр типа является контравариантным.

int — тип данных, хранящий целое число от -2 147 483 648 до 2 147 483 647 и занимающий 4 байта. Представлен системным типом System.Int32. Все целочисленные литералы по умолчанию представляют значения типа int.

interface — ссылочный тип, который может определять некоторый функционал - набор методов и свойств без реализации.

internal — является модификатором доступа для типов и членов типов.

is — проверяет, совместим ли тип среды выполнения для определённого выражения с указанным типом. Также ключевое слово is проверяет соответствие результата выражения указанному шаблону.

lock — получает взаимоисключающую блокировку заданного объекта перед выполнением определённых операторов, а затем снимает блокировку.

long — тип данных, хранящий целое число от –9 223 372 036 854 775 808 до 9 223 372 036 854 775 807 и занимающий 8 байт. Представлен системным типом System.Int64.

namespace — используется для объявления области действия, которая содержит набор связанных объектов.

new — создаёт экземпляр типа, а также может использоваться как модификатор объявления члена или ограничение универсального типа.

null — является литералом, представляющим пустую ссылку, которая не ссылается на объект.

object — тип данных, который может хранить значение любого из типов данных языка C# и занимает 4 байта на 32-разрядной платформе и 8 байт на 64-разрядной платформе. Представлен системным типом System.Object, который является базовым для всех других типов и классов .NET.

operator — используется для объявления оператора.

out — используется в качестве модификатора параметра, который позволяет передавать аргумент в метод по ссылке, а не по значению. Также ключевое слово out может применяться в объявлениях параметров универсального типа для интерфейсов и делегатов, что указывает на то, что параметр типа является ковариантным.

override — требуется для расширения или изменения абстрактной или виртуальной реализации унаследованного метода, свойства, индексатора или события.

params — используется для указания параметра метода, принимающего переменное число аргументов. Тип параметра должен быть одномерным массивом.

private — является модификатором доступа к члену.

protected — является модификатором доступа к защищённому члену.

public — является модификатором доступа для типов и членов типов с максимальными правами.

readonly — указывает на то, что присваивание значения полю может происходить только при объявлении или в конструкторе этого класса. Также ключевое слово readonly означает, что тип структуры является неизменяемым.

ref — указывает, что значение передаётся по ссылке.

return — завершает выполнение метода, в котором он присутствует, и возвращает управление вызывавшему методу.

sbyte — тип данных, хранящий целое число от -128 до 127 и занимающий 1 байт. Представлен системным типом System.SByte.

sealed — запрещает другим классам наследовать от этого класса.

short — тип данных, хранящий целое число от -32 768 до 32 767 и занимает 2 байта. Представлен системным типом System.Int16.

sizeof — позволяет получать число байт, занятых переменной заданного типа.

stackalloc — выделяет блок памяти в стеке.

static — используется для объявления статического члена, принадлежащего собственно типу, а не конкретному объекту. Также модификатор static можно использовать для объявления классов static.

string — тип данных, хранящий набор символов Unicode. Представлен системным типом System.String. Данному типу соответствуют строковые литералы.

struct — представляет собой тип значения, который может инкапсулировать данные и связанные функции.

switch — оператор выбора, который выбирает для выполнения один раздел switch из списка кандидатов, сравнивая их с выражением соответствия.

this — ссылается на текущий экземпляр класса, а также используется в качестве модификатора первого параметра метода расширения.

throw — сообщает о возникновении исключения во время выполнения программы.

true — правда.

try — блок, содержащий защищённый программный код, который может вызвать исключение.

typeof — получает экземпляр System.Type для указанного типа.

uint — тип данных, хранящий целое число от 0 до 4 294 967 295 и занимающий 4 байта. Представлен системным типом System.UInt32.

ulong — тип данных, хранящий целое число от 0 до 18 446 744 073 709 551 615 и занимающий 8 байт. Представлен системным типом System.UInt64.

unchecked — позволяет предотвратить проверку переполнения при выполнении арифметических операций и преобразований с данными целого типа.

unsafe — обозначает небезопасный контекст, необходимый для выполнения любых операций с применением указателей.

ushort — тип данных, хранящий целое число от 0 до 65 535 и занимающий 2 байта. Представлен системным типом System.UInt16.

using — создаёт псевдоним для пространства имён или импортирует типы, определённые в других пространствах имён. Также ключевое слово using определяет область, по завершении которой объект удаляется.

virtual — используется для изменения объявлений методов, свойств, индексаторов и событий и разрешения их переопределения в производном классе.

void — используется в качестве возвращаемого типа метода (или локальной функции) для определения того, что метод не возвращает значение.

volatile — означает, что поле может изменить несколько потоков, выполняемых одновременно.

while — выполняет оператор или блок операторов, пока определённое логическое выражение равно значению true.
